# Basic-Problem

This repository contains basic C++ problem-solving programs.  
These problems are useful for beginners who are learning programming logic, loops, conditions, functions, and number-based problems.

## 📌 Topics Covered

- Palindrome Number
- Digit Counting
- Reverse Number
- Sum of Digits
- Armstrong Number
- Prime Number
- Factorial
- Fibonacci Series
- Even or Odd
- Number Pattern Problems

## 🎯 Purpose of This Repository

The main purpose of this repository is to practice basic C++ programming concepts and improve problem-solving skills step by step.

## 🛠️ Language Used

- C++
